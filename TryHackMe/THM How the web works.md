# TryHackMe – How The Web Works

## Platform
TryHackMe

## Room
How The Web Works

## Topics Covered
- How the internet works
- DNS
- HTTP / HTTPS
- Web servers
- Requests and responses

## Key Concepts Learned
- DNS translates domain names into IP addresses
- HTTP is used for communication between client and server
- HTTPS encrypts communication using SSL/TLS
- Web servers host websites and respond to requests

## Practical Understanding
Learned how a browser sends a request to a web server and how the server responds with web content.
