## Topics Covered

### 1. What is a Network

A network is a group of devices connected together to share data and resources.

Examples:
- Internet
- Corporate networks
- Home networks

---

### 2. Types of Networks

LAN – Local Area Network  
Small network within a building.

WAN – Wide Area Network  
Large network spanning multiple locations.

MAN – Metropolitan Area Network

---

### 3. Devices in a Network

Router  
Connects different networks together.

Switch  
Connects devices within a LAN.

Firewall  
Filters and monitors network traffic.

---

### 4. IP Address

An IP address uniquely identifies a device on a network.

Example:
192.168.1.1

Two types:
- Private IP
- Public IP

---

### 5. TCP vs UDP

TCP
- Reliable
- Connection-oriented
- Used for HTTP, HTTPS, SSH

UDP
- Faster
- Connectionless
- Used for DNS, streaming

---

### 6. Common Ports

21 – FTP  
22 – SSH  
23 – Telnet  
25 – SMTP  
53 – DNS  
80 – HTTP  
443 – HTTPS  
445 – SMB



## Key Lessons

- Every device in a network needs an IP address
- Ports allow multiple services on one machine
- TCP ensures reliability while UDP prioritizes speed
- Understanding networking is critical for penetration testing
